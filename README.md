# 🚀 ExamNotesAI

An AI-powered SaaS platform to generate exam notes, diagrams, charts, and PDFs instantly using Gemini AI.

---

## 🔥 Features

✨ AI-generated exam notes in one click
📊 Auto-generated diagrams, graphs & charts
📚 Revision mode (short & crisp notes)
❓ Subject-wise important questions
💳 Credit-based usage system
💰 Stripe payment integration
🔐 Secure authentication (JWT)
🌐 Fully deployed (Frontend + Backend)

---

## 🛠️ Tech Stack

* Frontend: React.js (Vite + Tailwind CSS)
* Backend: Node.js + Express.js
* Database: MongoDB
* AI: Gemini API
* Payments: Stripe
* Auth: JWT
* Deployment: Vercel (Frontend), Render (Backend)

## 🚀 Installation & Setup

```bash
git clone https://github.com/manish5-k/ExamNotesAI.git
cd ExamNotesAI

# install backend
cd server
npm install

# install frontend
cd ../client
npm install

# run project
npm run dev
```
